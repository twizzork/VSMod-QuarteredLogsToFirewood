# twizzork's Quartered Logs to Firewood

This is a small recipe mod for Vintage Story that allows the user to chop up 1 quartered log into 4 firewood.

## Install

One click from the VS Mod DB, https://mods.vintagestory.at/tzquarteredfirewood

Or: drag & drop `tzquarteredfirewood` zip into your VS mods folder (win: `%APPDATA%\VintagestoryData\Mods`)

> [!NOTE]
> to find your mod folder: open VS to main menu, click "Mod Manager", then click the button at the bottom to "open mods folder"
